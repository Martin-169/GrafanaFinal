# GrafanaFinal

Mi grafan pose los siguientes dashboards:
Tengo 2 dashboards de stats donde en una muestro el mayor daño entre todos los pokemons y el nombre del pokemon. Y en el otro se ve el movimiento del pokemon que ha sido seleccionado su id en la variable creada.
Tengo 1 tabla donde se ven la altura, nombre y peso de cada pokemon de todo el JSON.
Tengo 1 pie chart donde se muestra el porcentage de los tipos que hay de cada pokemon.
Tengo 1 bar chart donde se muestra las estadisticas como ataque, defensa, velocidad y vida de cada pokemon enseñando solo el pokemon del id que se ha seleccionado.

Mi api es estatica al ser creada por mi en mockAPI.
Cree una variable llamada pokemonID en la cual cojo el id de cada pokemon para seleccionarlo y asi poder mostrar ciertas visualizaciones.
